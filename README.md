
# blog-web
블로그 예제를 통한 Spring Boot MVC, Spring Data JPA 학습
프로젝트의 기능은 후에 feature브랜치를 통해 점진적으로 추가하고 리펙토링 될예정

## 사용환경
- Java(JDK 11)
- IntelliJ IDE
- VScode (HTML, CSS 작업)
- MySQL 8.0.27
- Whimsical(와이어 프레임)
- ERDcloud
- Thymeleaf 템플릿 엔진을 통한 view 구성

## 현재 구현상태
- 블로그 회원가입, 로그인 처리 (시큐리티 미적용, PasswordEncoder를 통한 암호화만 사용)
- 게시글 조회, 생성, 수정, 삭제
- 마이페이지 게시글 관리 
<br/>

게시물 댓글 처리는 후에 기능 추가예정
